# Cube Combiner

This is a Blender Geometry Nodes setup that converts normal meshes into meshes made entirely of cube instances.

## Features

- Preserves materials  
- Control cutting algorithm with edge crease  
- Force cube directions with edge bevel  

## Installation

The geometry node setup is inside the given blender file.
